Projection Mapping Labs for Mable

At the end of last year, we have experimented for checking projected animation on Mable.
It project street vector data (SVG) with animation using CSS3 animation and D3.js.

We use awesome projector to project on a table.
It is SONY LSPX-P1 as Portable Ultra Short Throw Projector.

This animation make the street lines longer slowly.
Then it looks like putting light into streets.
